Prism

Prism is a differential privacy platform I built with Rails 8, PostgreSQL, and Python's differential privacy libraries to enable secure analysis of sensitive datasets. The system manages privacy budgets, validates queries for safety, and supports multiple cryptographic backends including DP, MPC, and homomorphic encryption. Added collaborative data rooms for multi-party computation scenarios, full audit logging, and a Python executor that adds calibrated noise to query results while maintaining formal privacy guarantees.
