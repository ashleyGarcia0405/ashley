Prism

Prism is a differential privacy platform I built with Rails 8, PostgreSQL, and Python's differential privacy libraries to enable secure analysis of sensitive datasets. The system manages privacy budgets, validates queries for safety, and supports multiple cryptographic backends including DP, MPC, and homomorphic encryption. Added collaborative data rooms for multi-party computation scenarios, full audit logging, and a Python executor that adds calibrated noise to query results while maintaining formal privacy guarantees.

Core Fellowship Platform

This platform brings applications, interviews, and placement decisions into one shared system, eliminating spreadsheets, scattered forms, and manual tracking. Interview feedback and preferences are recorded in real time, giving organizers a live view of the entire applicant pool and where each candidate stands. A guided matching tool then helps leadership quickly pair students with startups based on skills and preferences, while built-in messaging handles notifications and next steps automatically.

Autostack

A command-line assistant designed to automate environment setup by understanding what you’re trying to build and provisioning the tools you need. I once helped a friend (pre-med) set up a bio research workstation from scratch without access to the accumulated tooling most developers/CS students take for granted, and I thought it would be cool to have a tool scan local directories, infer missing dependencies, and generate step-by-step instructions or shell scripts to install languages, libraries, and tooling. This was about 2 years ago before Claude Code became public, so the project was an early experiment in AI-assisted developer ergonomics—and I’m currently revisiting and rebuilding it with better abstractions and stronger automation, informed by how the space has evolved over the last two years.

Aesthetic Mapping

An AI-driven fashion recommendation project I made with my friends at HackMIT that maps visual “vibes” from images to clothing items. The system analyzes an input image across structured dimensions using a multimodal LLM pipeline, then matches those attributes against a curated product database to surface aligned recommendations. The project was a bit of a hack job (API keys EVERYWHERE lol) but it was an early use of AI-integrated development environments back in 2023, so it was pretty cool to figure out tooling.

Gossip Girl

A deliberately terrible web app built at a Bad Ideas Hackathon hosted by The Tech Bros, where the goal was to design the worst apps possible. The application listens to live conversations in the browser, extracts dramatic or out-of-context snippets, and generates Gossip-Girl-style tweets via the OpenAI API. The project was intentionally invasive, unserious, and ethically questionable by design; it was a satire of surveillance tech, viral content pipelines, and how easily “ambient data” can be repackaged into performative drama.

Muselink

MuseLink is a distributed full-stack platform built as part of a Cloud Computing course to explore deploying services across multiple cloud providers. The application connects musicians with venues and individuals looking to book live performances, giving artists a way to showcase their work and sell their services directly. The system was architected as a set of independently deployed services, with frontend and backend components hosted across different cloud environments to study tradeoffs in latency, reliability, and operational complexity.
